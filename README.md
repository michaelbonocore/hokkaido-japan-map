# PetaPixel's Hokkaido Adventure

Interactive production planning map for Hokkaido, Japan.

## Overview

This project provides an interactive web map built with [Leaflet.js](https://leafletjs.com/) for planning and visualizing locations across Hokkaido, Japan.

## Features

- Interactive map interface
- Custom location markers
- Production planning tools

## Getting Started

Open `index.html` in a browser or visit the [live site](https://michaelbonocore.github.io/hokkaido-japan-map/).
